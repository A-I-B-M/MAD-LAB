This is a simple JavaScript task.
You need to Write "main.js" mentioned in the index.html. In the "main.js" 
you will need to add listeners to the appropriate button to calculate the 
BMI value that the user specified. Then show it on page with a little description. 

bmi <= 18.4, Underweight
bmi >= 18.5 && bmi <= 24.9, Normal
bmi >= 25 && bmi <= 29.9, Overweight
bmi >= 30, Obese.

You will also need to account for input errors.

You can modify the index.html as you see fit. You are not allowed to change the stylesheet.css.
