This is a simple JavaScript task.

need to add listeners to the appropriate button to calculate the BMI value that the 
user specified. Then show it on page with a little description.  

bmi = Weight (lbs) / [Height (inches)]² x 703

Your html will take the input in Metric system [Kg & m]. You can covert it to lbs & inches.1 m = 39.3700787402,
1 kg = 2.2046226218 lbs.

bmi <= 18.4, Underweight
bmi >= 18.5 && bmi <= 24.9, Normal
bmi >= 25 && bmi <= 29.9, Overweight
bmi >= 30, Obese.
